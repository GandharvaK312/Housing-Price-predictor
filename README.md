# Housing-Price-predictor  

Housing Price Predictor (Random Forest)

This project is a machine learning-based Housing Price Predictor developed in Python using the Random Forest Regression algorithm. The model is trained on historical housing data containing features such as location, area, number of bedrooms, bathrooms, and other property-related attributes. Random Forest, an ensemble learning technique, combines multiple decision trees to improve prediction accuracy and reduce overfitting.

The dataset is preprocessed by handling missing values, encoding categorical features, and splitting the data into training and testing sets. The trained model learns the relationship between property features and house prices, enabling it to predict the price of new properties based on user-provided inputs.

Key Features:

Data preprocessing and feature engineering
Random Forest Regression for accurate price prediction
Model evaluation using performance metrics such as R² Score and Mean Absolute Error (MAE)
Prediction of house prices based on custom input values
Implemented using Python libraries such as Pandas, NumPy, Scikit-learn, and Matplotlib

This project demonstrates the application of machine learning techniques in real-estate price estimation and provides a foundation for building more advanced property valuation systems.
